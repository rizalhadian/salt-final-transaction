### Get List Customer's Transaction
---
#### Endpoint :
```http
GET /api/customer/{customer_id}/transaction
```
#### Query Parameters :
| Parameter | Type | Description |
| :--- | :--- | :--- |
| `page` | `integer` | Page Transaction |

#### Response :
```javascript
{
    "success"                             : boolean,
    "message"                             : string,
    "page"                                : integer,
    "per_page"                            : integer,
    "page_count"                          : integer,
    "total_count"                         : integer,
    "data"            : [
        {
          "id"                          : integer,
          "total_price"                 : double,
          "total_discount"              : double,
          "total_price_after_discount"  : double,
          "note"                        : string,
          "created_at"                  : string,
          "status"                      : string,
        },
        {
          "id"                          : integer,
          "total_price"                 : double,
          "total_discount"              : double,
          "total_price_after_discount"  : double,
          "note"                        : string,
          "created_at"                  : string,
          "status"                      : string,
      },
    ]
}
```
